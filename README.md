## ğŸ¨ Customization

To customize the portfolio for your own use:

1. **Personal Information**: Update content in:
   - `src/components/Hero.tsx` - Name, title, and bio
   - `src/components/About.tsx` - About section content
   - `src/components/Contact.tsx` - Contact information

2. **Projects**: Edit the projects array in `src/components/Projects.tsx`

3. **Skills**: Modify the skills data in `src/components/Skills.tsx`

4. **Styling**: Customize colors and themes in `src/styles/globals.css` and `src/index.css`
